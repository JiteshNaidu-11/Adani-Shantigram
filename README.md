# Adani Shantigram – Township Marketing Site

## Getting started

Requirements:

- Node.js 18+
- npm

Install dependencies:

```sh
npm install
```

Start development server:

```sh
npm run dev
```

Build for production:

```sh
npm run build
```

Preview production build:

```sh
npm run preview
```

## Tech stack

- Vite
- React
- TypeScript
- Tailwind CSS
